# Angular 2 Meal Tracker

## Description
##### Meal Tracker. JS Code Review 2

## Setup/Installation

- clone repo
- npm install
- bower install
- install globals if needed (gulp, bower, sass, typescript, typescript packages.)
  - npm install gulp -g
  - npm install bower -g
  - gem install sass
  - npm install typescript -g
  - apm install atom-typescript
- gulp build
- gulp serve
